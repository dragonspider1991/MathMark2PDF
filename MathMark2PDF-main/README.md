# MathMark2PDF - Markdown & LaTeX Editor

A powerful markdown editor and converter with first-class support for mathematics and clean export options.

View your app in AI Studio: https://ai.studio/apps/drive/1ZRerbvTGXMf0odwFtSWZ6GlXFamYG9Lm

## Features

- **Rich Editor**: Syntax highlighting and LaTeX math support with KaTeX
- **Export Power**: Save as `.md`, `.html`, or print to `.pdf`
- **Split View**: Edit and preview markdown simultaneously
- **Theme**: Striking Black, Red, and Gold design
- **Math Support**: Inline and display math equations
- **Tables, Code Blocks, and More**: Full GitHub Flavored Markdown support

## Run Locally

**Prerequisites:**  Node.js (v16 or higher)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:3000`

## Build for Production

Build the app for production deployment:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Export Options

- **Markdown (.md)**: Export your document as a markdown file
- **HTML (.html)**: Export as a standalone HTML file with embedded styles
- **PDF**: Click the PDF button to open the browser's print dialog and save as PDF
